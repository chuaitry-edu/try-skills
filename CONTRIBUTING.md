# 如何贡献

感谢你愿意改进 try-skills 🧪

## 提 Issue

发现了 bug 或想加新功能？先搜一下有没有人提过。没有的话，开一个 Issue，说明：

- 你遇到了什么问题
- 你期望的行为
- 你用的是哪个 agent（Hermes / Claude Code / Codex / Cursor）

## 提 Pull Request

1. Fork 这个仓库
2. 改对应的 skill 文件
3. **注意同步**：hermes/ 下的 SKILL.md 是主版本。改完后，把 claude-code/、codex/、cursor/ 下的对应文件也改掉（内容一样，只是文件名后缀不同）
4. 提 PR，描述你改了什么、为什么改

## 目录对应关系

## 行为准则

保持友好。这是学习工具，不是战场。
