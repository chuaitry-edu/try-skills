# 学法匹配框架

> 出自 try-method skill

## 四类学习内容

| 你学的是 | 本质 | 怎么学 |
|---------|------|--------|
| **概念/知识** | 建立脑内模型 | 框架 → 回忆 → 输出 |
| **动手/技能** | 形成肌肉记忆 | 模仿 → 改参 → 独立 |
| **行为/习惯** | 重塑神经回路 | 环境 → 最小 → 扩大 |
| **感受/品味** | 校准审美标准 | 体验 → 对比 → 判断 |

## 新手最常见的错配

```
学编程（技能）→ 用看书的方法 → 崩溃
学哲学（知识）→ 用看视频的方法 → 空虚
改习惯（行为）→ 靠意志力 → 反弹
品酒（感受）→ 追求标准答案 → 无趣
```

## 时间紧怎么办

30 天以内：跳过基础期，直接做最小可用版本。
3 个月以上：按标准节奏走，前期多打基础。

## 混合理

如果目标混合了多种类型（如「做独立游戏」= 技能+知识），先选一种走。新手阶段，**动手比理解重要**。走完第一阶段觉得缺理论，再换方法。
