# 间隔重复实操指南

> 出自 try-review skill

## 核心原则

**分散学习 > 集中突击。** 同样 3 小时，分 3 天每天 1 小时，比一次性学 3 小时效果好一倍。

## 标准间隔

```
学完 → 24 小时 → 3 天 → 1 周 → 1 个月 → 每月
```

## 错过窗口怎么办

| 错过 | 怎么补 |
|------|--------|
| 错过 24 小时 | 从 3 天窗口开始，不要追 |
| 错过 1 周 | 直接重新学核心部分，这不是复习是重学 |
| 错过 1 个月+ | 比什么都不做好 |

## 不同内容的复习方式

- **概念/知识**：关书回忆 → 对照 → 重写补漏
- **动手/技能**：不看教程再做一次 → 查错
- **行为/习惯**：行为复盘 → 调整一个环境变量
- **感受/品味**：对比自己前后判断的变化

## 要不要用 APP

如果知识点多且量大（如背单词、背理论），可以用 Anki 等闪卡 APP 辅助管理间隔。工具帮你算时间，你只管回忆。
